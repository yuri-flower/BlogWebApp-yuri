3. What shall I go from here?

It totally depends on you and your interests

Any UI related programming has very similar logic to web programming and now you will be able to easily pick up them

Here is a list on what you can study if you would like to keep studying frontend:

  • Pick up a javascript framework and feel the modern design logic and code style: React, Vue, Angular
  • Pick up a state manangement framework and use it with the avascript framework you have picked: Redux, Vuex
  • Pick up a UI framework and learn how they architect their CSS and HTML: Boostrap, AntDesign, Material UI
  • Keep learning JS and learn about design patterns
  • Practice and make your creative ideas into real applications
  • Great places to learn: Mozilla guide, freeCodeCamp, The Modern Javascript Tutorial, JavaScript30
